## Описание

Этот проект содержит набор автоматизированных тестов для API `https://reqres.in` с использованием библиотеки `RestAssured` и фреймворка `JUnit 5`. Проект включает тесты для проверки основных функций API, таких как получение списка пользователей, создание и обновление пользователей, а также обработка неудачного входа в систему.

## Структура проекта

- **tests.ReqresInTests**: Основной класс с тестами. Содержит тесты для различных API-запросов к `reqres.in`.
- **tests.TestBase**: Базовый класс для настройки тестов. Здесь задается базовый URL и фильтры для логирования запросов и ответов.


## Запуск тестов
***Локальный запуск:***
```bash  
gradle clean test
```

## Тесты

1. **getUsersListTest**: Проверяет получение списка пользователей со страницы.
2. **createUserTest**: Проверяет создание нового пользователя.
3. **updateUserTest**: Проверяет обновление данных пользователя.
4. **unsuccessfulLoginTest**: Проверяет неудачный вход в систему без указания пароля.
5. **getUsersListWithDelayTest**: Проверяет получение списка пользователей с задержкой в 3 секунды.
